# Site de Receitas

## Objetivo

Desenvolver um site simples de receitas culinárias utilizando as principais tags HTML aprendidas em aula para estruturar corretamente as páginas.

O site será composto por quatro páginas interligadas, contendo uma página inicial e três páginas com receitas específicas.

---

## Estrutura do Projeto

O projeto será organizado com os seguintes arquivos:

- `index.html` → Página inicial com menu de navegação.
- `bolo-cenoura.html` → Receita de Bolo de Cenoura.
- `bolo-chocolate.html` → Receita de Bolo de Chocolate.
- `strogonoff.html` → Receita de Strogonoff.

---

## Requisitos Técnicos

O site deverá conter:

### 1. Listas
- Utilizar listas **não ordenadas (`<ul>`)** para ingredientes.
- Utilizar listas **ordenadas (`<ol>`)** para o modo de preparo.

### 2. Links
- Criar navegação entre todas as páginas.
- Permitir retorno para a página inicial.

### 3. Parágrafos
- Utilizar a tag `<p>` para descrever as receitas e adicionar informações complementares.

### 4. Imagens
- Inserir imagens ilustrativas das receitas utilizando a tag `<img>`.
- Incluir o atributo `alt` para acessibilidade.

###  5. Títulos
- Utilizar corretamente a hierarquia de títulos:
  - `<h1>` para o título principal da página
  - `<h2>` para seções
  - `<h3>` para subtítulos

## Publicação

O site deverá ser publicado utilizando o **GitHub Pages**.

### Entrega
Enviar o link público do GitHub Pages para acesso e avaliação.

## Tecnologias Utilizadas

- HTML5
- Estrutura semântica
- GitHub Pages para hospedagem

## Resultado Esperado

Um site funcional, organizado e navegável, demonstrando o uso correto das principais tags HTML aprendidas.
